# Image to PDF Guide (Step by Step)

This guide is written for users. 

## What this tool does

Turn one or many images into a single PDF file.

You can:
- choose paper size (A4, Letter, Legal, etc.)
- set background color
- add margins
- choose how images fit on the page
- reorder images before saving

## Quick flow

1. Open **Image to PDF**.
2. Upload your images.
3. Review **Page Setup** (A4 + White is default).
4. Reorder and rotate images if needed.
5. Preview PDF.
6. Save PDF.

## Full step-by-step

### Step 1: Open the tool

Go to the **Image to PDF** page from the Tools list.

### Step 2: Upload images

Use drag-and-drop or click **Add images**.

You can upload multiple files at once.

### Step 3: Set your page setup

Open **Page setup** and choose:

- **Page size**
  - Presets: A4 (default), Letter, Legal, Tabloid, A3, A5
  - Custom: enter Width + Height and unit (mm, cm, in)
- **Orientation**
  - Portrait or Landscape
- **Background**
  - White (default), Black, Gray, or custom color
- **Margins**
  - No margin, Small, Standard
- **Image layout**
  - Fit to page: keeps full image visible
  - Fill page: fills page, may crop edges
  - Original size: keeps image scale closer to original

### Step 4: Organize your images

In **Organize images**:

- drag to reorder pages
- rotate images (90Â° each click)
- remove unwanted images

Tip: Order here is the final page order in the PDF.

### Step 5: Check live preview

Use the live page preview panel to confirm:
- page shape/size
- margins
- image placement
- background color

### Step 6: Name your output file

In **Filename**, type your preferred file name.

`.pdf` is added automatically.

### Step 7: Preview before download (recommended)

Click **Preview PDF** to open a full PDF preview dialog.

### Step 8: Save your PDF

Click **Save PDF** to download the final file.

## Common use cases

- **Scanned contracts**: A4 + White + Standard margin + Fit to page
- **Photo pages**: Letter + Fill page
- **IDs or custom print sizes**: Custom page size + small/no margin

## Helpful notes

- Processing happens in your browser.
- Your setup choices are remembered for next time on the same browser.
- If something looks off, open **Page setup**, adjust, then preview again.
